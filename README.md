# Lucci - OpenIntents
Lucci is an open-source Typescript SDK and AI interface that transforms natural language prompts into complex multi-chain DeFi workflows. Build, execute, and scale the next generation of crypto agents.

## Packages
1. **@lucci/core**: The core SDK for building and executing DeFi workflows.

## Installation
To install the Lucci SDK, use npm or yarn:
```bash
npm install @lucci/core
# or
yarn add @lucci/core
```
